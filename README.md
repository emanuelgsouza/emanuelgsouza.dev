## Technologies that I use

* [Vuepress](http://vuepress.vuejs.org)
* [Tailwind](https://tailwindcss.com)

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:8080
$ yarn run dev

# build for production and launch server
$ yarn run build
```
